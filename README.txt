A watch with HPDL-2416 display and ATmega16 microcontroller.

Schematic and PCB is designed using Eagle. AVR code project created using Atmel Studio 6.

Don't forget to set right fuses when programming your AVR. Fuse bits for khazama AVR Programmer: \HPDL_watch\AVR code\Fuse bits.PNG
